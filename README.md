> Verb plugin that adds commonly needed data to the context for rendering templates.

## Install

Install with [npm](https://www.npmjs.com/):

```sh
$ npm install verb-repo-data --save
```

## Usage

```js
var data = require('verb-repo-data');

// register the plugin
app.use(data());
```
